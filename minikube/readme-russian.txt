1) Для запуска Kubernetes кластера с помощью Minikube просто выполните команду:

minikube start

#------------------------------------------------------------------------------

2) Для остановки кластера наберите команду:

minikube stop

#------------------------------------------------------------------------------

3) Для просмотра текущего статуса кластера наберите:

minikube status
